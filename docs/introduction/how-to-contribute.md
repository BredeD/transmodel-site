# How to contribute

The Transmodel family of standards is maintained under [CEN/TC 278 Working Group 3](governance.md) — Public Transport. Contribution paths depend on what you want to influence.

## Contribution paths (to be detailed)

- **Raising a change request** against an existing standard (NeTEx, SIRI, OJP, OpRa) — via the relevant CEN sub-group's process.
- **Contributing to a national profile** — via the national profile's own process (Nordic, French, Italian, etc.).
- **Contributing to this site** — see the repository's `WASH-GUIDE.md`.
- **Joining a NAPCORE working stream** — via [NAPCORE](napcore.md).
- **Contributing to the parallel [Guides project](https://github.com/TransmodelEcosystem/NeTEx-Guides-Documentation)** — for practical implementation guides.

## What to read first, depending on interest

- Raising a change request → the standard's own repository under [TransmodelEcosystem](https://github.com/TransmodelEcosystem).
- Improving a national profile → the profile's documentation and its issue tracker.
- Coordinating with other Member States → NAPCORE working streams.
