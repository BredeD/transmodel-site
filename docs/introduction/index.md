# Introduction

Welcome to the entry point for the **Transmodel Ecosystem** — the family of European public transport data standards. This section is where newcomers, decision-makers, and anyone considering adoption of these standards should start.

CEN publishes the specifications. Vendors write product documentation. National access points publish their own profiles. This site — and this section in particular — sits between: explaining what the standards are, who runs them, how they fit together, and how to get started with them, without duplicating any of those layers.
